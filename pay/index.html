# 支付

调用App的支付Api，完成第三方支付功能，如：微信支付，支付宝支付等。

# 支付宝支付

# 参数

属性 类型 默认值 必填 说明
data string 是 预支付订单信息
paySuccess function 否 支付成功回调
payFail function 否 支付失败回调

# 示例代码

预支付订单信息由后端提供。

import { alipay } from 'flutter-app-jssdk';

alipay(
  data: 'xxxxxxxx',
  () => {
    console.log('支付成功');
  },
  () => {
    console.log('支付失败');
  },
);

# 微信支付

# 参数

属性 类型 默认值 必填 说明
data Object 是 预支付订单信息
paySuccess function 否 支付成功回调
payFail function 否 支付失败回调

# 示例代码

预支付订单信息由后端提供。

import { wechatPay } from 'flutter-app-jssdk';

wechatPay(
  data: {
    appId: 'appId',
    partnerId: 'partnerId',
    prepayId: 'prepayId',
    packageValue: 'packageValue',
    nonceStr: 'nonceStr',
    timeStamp: 'timeStamp',
    sign: 'sign',
  },
  () => {
    console.log('支付成功');
  },
  () => {
    console.log('支付失败');
  },
);

# 支付弹窗

可以呼起App端的支付弹窗，然后通过传参将获取预支付信息的接口地址传入支付弹窗，由支付弹窗统一处理。

# 参数

属性 类型 默认值 必填 说明
data Object 是 弹窗参数
success function 否 支付成功回调
fail function 否 支付失败回调

# 示例代码

import { showPayPop } from 'flutter-app-jssdk';

showPayPop(
  data: {
    total: '100', // 支付金额
    orderNo: 123456, // 支付订单号
    payApi: 'https://xxxxpayxxxx',
  },
  () => {
    console.log('支付成功');
  },
  () => {
    console.log('支付失败');
  },
);
