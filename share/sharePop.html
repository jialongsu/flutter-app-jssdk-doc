# 分享弹窗

呼起App端的分享弹窗

# 参数

属性 类型 默认值 必填 说明
data Array 是 需要在弹窗中显示的分享渠道数据
success function 否 分享成功回调

# data详细参数

属性 类型 默认值 必填 说明
channel number 是 呼起分享弹窗时自定义显示的分享渠道 1：新浪微博 2：微信好友 3：朋友圈
shareData Object 是 分享的数据

# shareData详细参数

属性 类型 默认值 必填 说明
shareType string 是 分享类型: mini（小程序） news（网页） image（网络图片） text（文本） video（视频） audio（音频）
title string 否 分享标题
description string 否 分享描述
webUrl string 否 分享网页地址
image string 否 分享图片地址
videoUrl string 否 分享视频的地址
musicUrl string 否 分享音频的地址
miniPath string 否 分享小程序的地址

# 示例代码

import { sharePop } from 'flutter-app-jssdk';

// 分享弹窗会显示微信好友与朋友圈两个分享渠道
sharePop(
  data: [
    {
      channel: 2, // 微信好友
      shareData: {
        shareType: 'news',
        webUrl: 'http://www.baidu.com',
        image: 'http://xxxx.png',
        title: '分享标题',
        description: '分享描述'
      }
    },
    {
      channel: 3, // 朋友圈
      shareData: {
        shareType: 'news',
        webUrl: 'http://www.baidu.com',
        image: 'http://xxxx.png',
        title: '分享标题',
        description: '分享描述'
      }
    }
  ],
  () => {
    console.log('分享成功');
  },
);
