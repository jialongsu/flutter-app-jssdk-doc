# 选择相册图片

通过pickerImage可以打开本地相册选取照片，选取成功返回图片的base64值。

# 参数

属性 类型 说明
maxImages number 限制选取图片的数量
limitSize number 限制选取图片的大小，默认5M

# 示例代码

import { pickerImage } from 'flutter-app-jssdk';

// 最多选2张图片，限制每张大小为10M
const imgs = await pickerImage(2, 10); 
console.log('imgs:', imgs);
