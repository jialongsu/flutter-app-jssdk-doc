# 下载图片到本地

通过downloadImage可以将图片保存到本地。

downloadImage支持两种图片数据类型：

  • 直接传入网络图片链接
  • 传入base64图片数据

# 参数

属性 类型 说明
image string 网络图片链接
base64Image string base64图片数据

# 示例代码

import { downloadImage } from 'flutter-app-jssdk';

// 下载网络图片
downloadImage({image: 'http://xxxxx.png'}); 

// 下载base64图片
downloadImage({image: 'base64xxxxx'});
