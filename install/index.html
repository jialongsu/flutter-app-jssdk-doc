# 安装

使用yarn

yarn add flutter-app-jssdk

或使用npm

npm add flutter-app-jssdk

或使用pnpm

pnpm add flutter-app-jssdk

# 使用

使用api前需要需要在项目挂载完成后，调用injestJs方法：

import { injectJs } from 'flutter-app-jssdk';

injectJs();

该方法是为了注入第三方jssdk，如微信的jssdk，用于调用微信的桥接api，用于与公众号或者小程序通信。

注：该方法越早调用越好，由于会请求远程的js文件，避免出现在使用到api时js文件未加载完成的问题。
